### 실습 내용

#### 설정 명령어
- git config
- .gitignore

#### 확인 명령어
- git log
  - --oneline
  - --graph
  - --stat
- git status / diff
- git reflog

#### 기본 명령어
- git clone
- git add 와 commit
- git push
- git pull 와 fetch

#### 돌아가기
- git revert
- git reset
  - --soft / --mixed / --hard
 
#### 브랜치 관련
- git branch
- git switch / checkout
  - git switch -c / -d / -D
  - git checkout HEAD^^^ == git checkout HEAD~3
- git rebase
  - --continue
  - --abort
- conflict 해결
- git cherry-pick
